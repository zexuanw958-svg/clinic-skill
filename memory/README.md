# Clinic Memory

This directory contains templates for `clinic`'s auditable memory system.

The repository only stores empty templates. Personal memory should live in the user's local skill directory, for example:

```text
/Users/wzx/.agents/skills/clinic/memory/
```

Rules:

- Do not store secrets or sensitive identifiers.
- Do not store hidden memory without user confirmation.
- Treat every memory item as a revisable hypothesis, not a permanent label.
- Prefer high-signal patterns over raw transcripts.
